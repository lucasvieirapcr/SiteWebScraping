### Próximo Passo: Envie para o GitHub

Agora que você tem o arquivo, precisa adicioná-lo ao seu repositório para que a plataforma de hospedagem possa vê-lo.

1.  **No terminal (com o Git já iniciado), adicione o novo arquivo:**
    ```bash
    git add requirements.txt
    ```

2.  **Faça o commit:**
    ```bash
    git commit -m "Adiciona arquivo de dependências (requirements.txt)"
    ```

3.  **Envie para o GitHub:**
    ```bash
    git push
    ```

Agora seu repositório está completo e pronto para o deploy
